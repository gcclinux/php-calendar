# php-calendar
PHP Calendar

* Install apache2
* Install php7.4
* Install php7.4-curl
* Install php7.4-mysql or php7.4-pgsql

* Restart apache2 
* Download tar file
* tar xvf into webserver root or subfolder


Wiki - https://github.com/gcclinux/php-calendar/wiki
